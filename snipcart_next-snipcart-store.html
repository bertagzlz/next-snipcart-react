Next.js Snipcart Store Demo

This is a demo project using Snipcart to create an ecommerce store with Next.js.

Getting Started

Environment File

Create a file called .env.local at the root of the project with:

NEXT_PUBLIC_SNIPCART_API_KEY="[Your Snipcart API Key]"

Running the Project

Start your development server:

npm run dev
# or
yarn dev

Open http://localhost:3000 with your browser to see the result.
